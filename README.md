# go-doom-fire

Implementação em Go do clássico efeito de fogo do Doom, renderizado no terminal.

![Dooom fire Demo](./assets/doom-fire.gif)

## Como executar

```bash
go run cmd/main.go
```

## Atalhos

- `Esc`: sair
- `Ctrl+C`: sair
